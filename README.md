# ArchipelagoIV
Archipelago randomizer world for Grand Theft Auto 4 / Episodes From Liberty City. Designed for use with the AP client repo. 
Only supports Patch **1.0.8.0** and **1.0.7.0**

## THIS DOES NOT INTERACT WITH ROCKSTARS SERVERS IN ANY WAY, SHAPE OR FORM. THIS IS A SINGLE PLAYER MOD
Note this is alpha stages right now so please expect a lot of bugs or missing features.


## How to run
### Required Software

- [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases/latest)
- Any GTA IV Downgrader that can downgrade to **1.0.7.0** or **1.0.8.0**
- [IVSDK .NET](https://github.com/ClonkAndre/IV-SDK-DotNet)
- [ClonksCodingLib.GTAIV](https://github.com/ClonkAndre/ClonksCodingLib.GTAIV)
- [Base Game](https://github.com/guesttest968/ArchipelagoIV/releases) or [TBoGT (The Ballad of Gay Tony)](https://github.com/guesttest968/ArchipelagoIV/releases) APWorlds
- [ArchipelagoIV](https://github.com/guesttest968/ArchipelagoIV/releases)

### Install Mod

First downgrade your game if you have the complete edition to patch 1.0.7.0 or 1.0.8.0
Install IVSDK .NET and ArchipelagoIV to your game files and ClonksCodingLib to `rootgame\IVSDKDotNet\bin`
From there you can launch the game and press F3 to open the ArchipelagoIV menu

# What Episodes does this support?
For now only TBOGT and Base Game are supported. 


# Why is CE (Complete Edition) not supported?
Due to lack of support for CE in IVSDK .NET Complete Edition is basically out of the water unless a better method or have support for it helps
